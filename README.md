## Installation

### Chrome/Edge/Brave
1. Download the latest release from [Releases page]
2. Unzip the package
3. Navigate to `chrome://extensions`
4. Enable "Developer mode" (toggle in top-right)
5. Click "Load unpacked" and select the unzipped folder
